# React + TypeScript + Styled Components
Boilerplate for a project using React and Typescript

## Installation

Use a package manager of your choice (npm, yarn, etc.) in order to install all dependencies

```bash
npm install
```

```bash
yarn install
```

## Usage

```bash
npm run start
```

```bash
yarn start
```
